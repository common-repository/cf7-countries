Country drop-down menu for Contact Form 7.

== Description ==

Country drop-down menu for Contact Form 7. Install the plugin and you will get "countries drop down" Form tag in CF7. The countries list is the same countries list used by WooCommerce.

== Installation ==

1. Upload `cf7-countries.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for "countries drop down" form tag in Contact Form 7
